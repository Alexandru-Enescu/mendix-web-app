# Web application built with Mendix

## Project description
This is the website of a fictive electronics company. The website helps customers to fix their broken devices or allows customers to submit requests for repairment.

## Steps for running the app
! Studio Pro IDE is required
1. Download the mpk file
2. Extract project files to a directory
3. Studio Pro IDE should be automatically opened
4. Click on the 'Run' button to run the project locally
5. Click on the 'View App' button to open the project in the browser
6. Log in to the website
     Username: MxAdmin
     Password: 1
7. In the right side is a button which allows to change the type of user (view the website from a different perspective).
   There are 3 user types: Customer, CustomerService and RepairmentService
